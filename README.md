 # Full-Stack Express Authentication System
    
     A modern, full-stack authentication application built to practice the fundamentals of Node.js, Express, and React integration. This project demonstrates how to
      handle user data securely from the frontend to a backend API.
    
## 🚀 Features
     - **User Signup:** Securely register users with encrypted passwords.
     - **Form Validation:** Frontend checks for empty fields and matching passwords.
     - **RESTful API:** Express backend with endpoints for user management.
     - **Security:** Industry-standard password hashing using Bcrypt.
    - **CORS Enabled:** Seamless communication between the Vite frontend and Express backend.
   
## 🛠️ Tech Stack
   
### Frontend
    - **React (Vite):** Fast, modern UI library.
    - **Tailwind CSS:** Utility-first styling for a clean, responsive design.
    - **React Router:** For seamless navigation between Login, Signup, and Home pages.
   
### Backend
    - **Node.js & Express:** Robust server-side framework.
    - **Bcrypt.js:** To securely hash and salt passwords before storage.
    - **CORS:** To allow cross-origin requests from the frontend.
    - **Nodemon:** For an improved development workflow with auto-restarts.
   
## 📖 What I've Learned
    Through this project, I have mastered several key concepts in full-stack development:
    - **Backend Architecture:** Setting up an Express server and defining API routes.
    - **Data Processing:** Using `express.json()` to parse incoming request bodies.
    - **Asynchronous JavaScript:** Using `async/await` and `fetch` to handle API calls.
    - **Security Best Practices:** Why we never store plain-text passwords and how `bcrypt` salts/hashes data.
    - **Cross-Origin Resource Sharing (CORS):** Bridging the gap between different ports during development.
    - **Component-Based UI:** Building reusable components like `Input`, `Button`, and `AuthForm`.
